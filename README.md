<h3>课程介绍：</h3>
<p>第一阶段：会实现电商系统的大部分功能，包括：用户管理，订单管理，商品管理，支付功能，后台管理系统，单点登录，购物车，搜索功能，个人中心等等。</p>

<p>第三阶段：引入大数据分析，加入防爬虫功能，采用多语言构建我们的微服务，比如：Python，Go语言等。</p>
